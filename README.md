# Semantic HTML5 Portfolio Website

A fully accessible multi-page personal portfolio website built using semantic HTML5, ARIA attributes, and SEO best practices.

## Features
- Semantic HTML5 structure
- WCAG accessibility compliance
- ARIA labels and roles
- SEO-friendly meta tags
- Accessible contact form
- Keyboard navigation support

## Pages
- Home
- About
- Contact

## Technologies Used
- HTML5
- Accessibility (WCAG)
- ARIA
- SEO Best Practices
